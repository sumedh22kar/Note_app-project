# 📝 Note App

A simple Note App built with **React.js** and **Tailwind CSS**.

This project allows users to add notes by entering a title and details. Added notes are displayed in the "Recent Notes" section, and users can delete any note with a single click. The application uses React's `useState` hook for state management and features a responsive layout for desktop and mobile devices.

## Features

- Add new notes
- Delete existing notes
- Responsive layout
- Built using React functional components
- State management with `useState`

## Technologies Used

- React.js
- JavaScript (ES6)
- Tailwind CSS
- HTML
- CSS

## Installation

```bash
git clone https://github.com/sumedh22kar/Note_app-project.git
cd Note_app-project
npm install
npm run dev
```
